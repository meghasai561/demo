# BankNifty Options Trading Bot

This is a production-ready trading bot for NSE BankNifty Index Options using Angel One SmartAPI.

## Features
- Automatically fetches the current BankNifty options expiry.
- Monitors the first 30-minute candle after market open.
- Sets high and low levels from the first candle.
- Monitors 3-minute candles thereafter.
- Sells PUT options (1 lot) if a 3-min candle closes above the 30-min high.
- Sells CALL options (1 lot) if a 3-min candle closes below the 30-min low.
- Manages positions with 10 points target and 13 points stop loss.
- Exits positions by 3:15 PM if target/stop loss not hit.
- Supports both paper trading (demo) and live trading.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. **Generate Angel One Credentials** (see below).
3. Update the configuration in `main.py`:
   - Set `IS_PAPER = True` for demo/paper trading, `False` for live.
   - Replace credentials.
4. Run the bot: `python main.py`

### Generating Angel One Credentials
To use the Angel One SmartAPI, follow these steps:

1. **Sign Up for Angel One Account**:
   - Visit [Angel One website](https://www.angelone.in/) and create a trading account if you don't have one.
   - Complete KYC and enable trading permissions.

2. **Access SmartAPI**:
   - Log in to your Angel One account.
   - Go to the "SmartAPI" or "Developer" section (usually under "More" or "Tools").
   - If not visible, search for "SmartAPI" in the Angel One app or website.

3. **Create an API Key**:
   - In the SmartAPI dashboard, create a new app or API key.
   - For demo/paper trading, select the demo environment.
   - For live trading, use the live environment.
   - Note down the `API_KEY` provided.

4. **Get Client ID and Password**:
   - Your `CLIENT_ID` is your Angel One client ID (visible in your account profile).
   - `PASSWORD` is your Angel One trading password (set in account settings).

5. **Set Up TOTP (Two-Factor Authentication)**:
   - In your Angel One account, enable TOTP if not already done.
   - Use an authenticator app like Google Authenticator or Authy.
   - Scan the QR code provided by Angel One.
   - The `TOTP_PIN` is the 6-digit code generated by the app at runtime. You can either:
     - Manually enter it each time, or
     - Implement code to generate it programmatically (e.g., using `pyotp` library).

6. **Update `main.py`**:
   - Replace the placeholder values with your actual credentials.
   - For paper trading, set `IS_PAPER = True`.
   - For live, set `IS_PAPER = False` and ensure you have funds.

**Note**: Keep your credentials secure. Do not share them. For paper trading, use demo credentials to test without real money.

## Important Notes
- This bot trades live or paper; use at your own risk.
- Ensure you have sufficient funds and permissions for options trading.
- BankNifty lot size is 25.
- Strikes are rounded to nearest 100.
- The code aggregates ticks into 30-min (first) and 3-min candles.
- Handle errors and disconnections properly in a real setup.
